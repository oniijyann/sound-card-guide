1前言

麦克风和声卡设计的目标不是为了把声音变好听，而是为了更精准地还原声音，获得更好的信噪比。让你声音有变化的是音频处理软件（DAW）里使用的各种插件。

声卡调试不能让你声音有非常大的改变，也不能让你演唱的节奏和音准更好，更不能让你唱歌能和某某一样。声卡调试的主要工作内容是能让小白快速获得一份自己比较满意的声音。

B 站是个巨大的知识宝库，喜欢深入研究的找找教程，你也可以调出自己想要的声音。

2调试前沟通

💰明确调试费用

明确调试声卡的费用，确认是否有额外收费，以及后续问题如何解决。

🎵明确直播内容类型

了解客户的直播内容类型：唱歌、乐器演奏、唱跳等。这主要为了后续麦克风和声卡选择做参考。

🎤明确声卡和麦克风型号
  • 有些娱乐声卡不支持 ASIO 驱动，那些直接用声卡自带的效果就好，不需要调。
  • 方便找到对应的声卡驱动，还可以提前找到产品图，方便后续声卡和麦克风的基本使用教学。
  • 如需多个麦克风或有乐器（电钢琴、电吉他），要注意声卡是否有相应的接口。
🔌明确其他配件是否齐全

如果是用手机直播的，需要声卡有 OTG 接口加 OTG 线。

  • 没有 OTG 接口的声卡需要音频转换器（直播一号之类的产品）。
  • 注意 OTG 线接充电器最好用原装充电器，不要接那种很多插口的分线器。
  • 最好直接在买声卡的店铺买 OTG 线，或者问清楚店家是否有特殊要求，不兼容普通的 OTG 线。

建议在声卡调试前就安装好。

确认卡农线齐全，能正常连接声卡与麦克风。

  • 注意有些声卡是 6.35 接口的，对应 3.5 接口的耳机需要个转接头（一般买声卡都会送）。
  • 如果声卡距离人比较远，还需要个耳机延长线。
  • 不建议使用无线耳机：有延迟，且需要充电。

确认好声卡与电脑的接口：

  • USB
  • PCIE
  • 雷电
  • 注意 USB 接口建议用独立的 USB，不要使用 USB 集线器与其他设备共用。
  • 不同声卡的接口不同，需要电脑有对应的接口和对应的连接线。

只要不是很久之前买的电脑，且是 Win10 以上的电脑都可以。

🕐约定调试时间

约定调试时间，并告知客户大概需要调试多久。

3调试：软件安装与配置

1 设备连接

指导客户将声卡连接电脑和麦克风。一般声卡详情页或者说明书都有，看一下都可以完成的。

注意：声卡不要放在电脑机箱上，也尽量远离各种电子设备，离太近可能会造成干扰。关闭声卡上的硬件监听。

注意麦克风类别：

  • 动圈麦
  • 电容麦——需 48V 供电
  • 无线麦要注意接收器频段是否与麦克风相符，有时频道不匹配也会有声，但有杂音。

如果有乐器或者话放等设备，需注意接口的电平级。

2 远程控制

请客户下载远程控制软件（ToDesk 或者向日葵），安装后打开软件，将软件显示的号码发给调试人员。

调试人员远程连接控制电脑，下面操作交给调试人员就好了。

3 驱动安装
  • 如之前已经装过声卡驱动且有调试过，需备份之前的文件，方便后期需求回到之前的状态。
  • 去声卡官网找到对应驱动并安装（有的声卡安装驱动后需要重启电脑）。
  • 打开声卡驱动面板，一般将声卡采样率调整至 44100，缓冲区大小调整至 128（电脑比较好的可以调更小）。
  • 声卡上麦克风（电容麦）增益旋钮调整至 12 点位置，并且确认麦克风轨道有波形即可。
  • Windows 声音设置里各个通道都改成 44100 采样。改采样率旁边有个测试的按钮，可以点一下以确认客户耳机可以听到声音。
  • Windows 声音设置里"播放"选项卡中，将默认通道设置为声卡的硬件通道（耳机）。
  • 取消独占模式和各种增强模式。
  • Windows 声音设置里"录制"选项卡中，将默认通道设置为声卡声音输出的通道（不同声卡情况不同）。
  • Windows 声音设置—"通信"选项卡里改成"不执行任何操作"。
  • 禁用掉其他无关的通道。
4 软件安装

询问客户需要什么音乐软件。常用音乐软件有 QQ 音乐、全民 K 歌、酷狗音乐等等。然后就可以开始 DAW、音乐软件、各种插件的安装和设置，这段时间可以请客户去休息 10-20 分钟。

通过百度网盘或者远程软件把插件和 DAW 安装包下载到电脑里，并安装。

  • Studio One
  • Cubase
  • MonsterAudio（免费）

列举一些比较常用的插件：FabFilter、Waves、Valhalla、板岩、恐龙、插件联盟。

关键原则：什么插件其实无所谓，注意要选 0 延迟或者低延迟的，不然监听有延迟会影响唱歌。

下载安装客户需求的音乐软件，安装完成后需设置到对应的伴奏轨道。

安装 YY 或者 Sonobus（用于远程监听客户声音）。

5 音频信号路由

声卡音频信号路由

  • 不同声卡多少都有些区别，需根据具体声卡去设置。
  • 新建一个或者打开制作好的 DAW 工程文件模板（如有）。
  • DAW 中路由设置：就是把声卡的通道连接进 DAW 处理后再输出。
检查项：注意检查"高级选项 → 控制台 → 提示混响静音跟随通道"勾选上。

音乐软件播放通道设置

  • 将播放通道设置到与 DAW 伴奏通道一致。
  • 全民 K 歌有通道独占的问题，建议声卡有多余通道的，单独给它设置一条通道。且全民 K 歌中的麦克风通道要单独设置一路，不要与主输出通道一路，因为它会降低到 80% 的音量。并且在 Windows 音频设备播放通道选择对应通道。

直播软件通道设置

直播软件的麦克风设置选从 DAW 的总输出通道。

直播软件的扬声器：

  • 一般直接选声卡的耳机硬件通道。
  • 通道多的，也可以单独在 DAW 里建立一条轨道，不发送给总输出，只发送给耳机。
6 电脑优化设置
  • 设备管理器中把 USB 根集线器设备休眠禁用。
  • 电源方案要选高性能，并且把"USB 设置 → USB 选择性暂停设置"设为禁用；"硬盘 → 在此时间后关闭硬盘"设为 0 分钟。
  • 电源选项里，找到"定义电源按钮并启动密码保护"，里边有个"启用快速启动"，这个要关掉！尤其笔记本用户。
  • 如还有噼啪声，可以考虑把声卡驱动和 DAW 的运算优先级调到"实时"（可使用 Process Lasso 一直设置），或者增大声卡缓冲区大小。
  • Win11 系统在 Studio One 6 上可能出现杂音，需用 Process Lasso 或 CoreDirectors 把程序设置到只在大核上运行。
  • 如上述设置还不能避免噼啪声，那还要根据电脑硬件去寻找原因。比较新的声卡和电脑一般不会有这个问题。

完成上述设置后就可让客户回来继续。

4简单的设备知识教学

🎤 麦克风基础

  • 介绍心形指向。如果麦克风上有功能开关，可结合产品图片告知客户其功能并指导选择正确位置。常见开关有：底切开关、指向性选择、衰减开关。
  • 嘴一般正对麦克风振膜。
  • 嘴与麦克风距离建议 15-25cm（两拳）。
  • 如嫌弃挡脸，可把麦克风的振膜对准嘴的前提下，移动到其他位置也可。
  • 风扇和空调不要对着麦克风吹。
  • 麦克风尽量远离噪声源，利用心形指向避开直接对着噪音源。

🎚️ 声卡基础

  • 告知麦克风增益旋钮是哪个。
  • 告知耳机音量旋钮是哪个。
  • 告知硬件监听是哪个。
  • 一般声卡详情页或者说明书都有图片说明。

5试唱与调试

试唱流程

通过 YY 或者 Sonobus 监听客户从 DAW 总线出来的声音。整个试唱过程通过多轮迭代，逐步调整音量和效果。

第 1 轮 初次试唱与问题排查
  • 电容麦建议把麦克风增益扭到 12 点钟方向。
  • 一般大声说话时电平能冲到 -6dBfs，但不超过 0；不说话时底噪不超过 -48dBfs。
  • 让客户咳一下，电平表没超过 0 就行（每个声卡的最大输入电平不一样，如果音量较小，后边使用数字增益增大音量即可）。
  • 如可以，也可以让客户先开嗓。
  • 让客户带伴奏唱一小段副歌，几句即可。请客户随时调整耳机音量来听清自己声音，能正常舒适地唱下来即可，不需要很精确。
  • 唱的时候需录下来。
  • 可以适当加少量混响。
听录音判断是否有明显的问题并解决：
  • 声音过大或过小
  • 是否有明显的底噪
  • 是否有近讲效应
  • 房间反射声特别大
  • 最基本的原则是不能爆音，且应有部分余量
第 2 轮 录制与初步调试
  • 通过前边的调整和问题解决后，录制一首熟悉的歌，只录一遍主歌加副歌即可。
  • 然后根据录好的歌进行调试。
  • 经过调整后，整体音量会变大。请客户调整声卡上耳机音量旋钮，调到音量刚好合适即可，不需要很精确。
  • 听录音，根据客户的反馈进一步调整。
第 3 轮 带效果验证与反馈
  • 请客户带着刚才调试好的效果再唱一首（最好是同类型的歌曲）。
  • 此次演唱需要客户根据人声在耳机中的大小去调整耳机音量，可以一边唱一边调，目标是能正常舒服地演唱。
  • 同时可以询问混响大小是否合适，是否会影响演唱。混响过大可能会影响听不清自己声音。
  • 客户可能无法对于效果说出明确的要求，但对于人声、伴奏、混响的音量一般都可以表达出是大了还是小了的反馈，所以可以多试试不同音量，让客户给出反馈。
  • 听录音，根据客户的反馈进一步调整。
第 4 轮+ 多曲验证与最终确认

可以多试几首歌。声音效果是一方面，另外一方面是需要演唱者在演唱过程中听自己的音量刚好、听着舒服，能发挥出正常水平即可。

补充 其他调整

如希望演唱时耳机监听人声大一些，但直播间输出时候不需要那么大（或反之），则可调整耳机中人声和伴奏的音量，来达到演唱中耳机声音比例听着舒服，且输出给观众的音量比例也合适。

DAW 效果调试

按照轨道类型分别调试，以下为各轨道的处理建议。

🎙️ 人声轨

降噪

原则上越少越好，如果听不到明显底噪的可以不加。常用插件有 NS1、PSE（门限）、RX Voice.De-noise、Bertom.Denoiser。

注意：如果是乐器，请不要使用降噪。

修正音高（一般不用）

使用 Autotune，要注意选低延迟模式。这个东西唱得好的不需要，音特别不准的它也没用。

EQ 修复调整

一般会加个低切，然后适当修复一些频段。常用插件如 FabFilter Q3。

压缩

对于直播的场景来说，只建议做控制动态的压缩。常用插件有 2A、76、RVOX、CL1B、Cenozoix.Compressor、670、API2500。

其他——增加谐波染色

带有染色功能的 EQ 或者其他插件。如 1073 的模拟话放染色、EQP-1A、Fresh Air、Roth-AIR、Maag EQ4、Black.Box.Analog、Saturn 2、SSL 通道条。

去齿音

如 FabFilter DS、sibilance。

其他特殊效果

如需要其他特殊效果，对应添加需要的效果器插件即可。如果没有特定类型的歌曲的需求（复古老歌、美声等），一般都以声音干净均衡为目标进行调整。

🌊 发送空间效果轨

混响

人声一般可以用板式混响加大堂混响。常用插件有 Valhalla VintageVerb、pro-r、Gemini BM7、第七天堂。

混响上可以加大范围去齿音插件，来减少混响特别大的听感。

高低切调整

高切可改善声音的亮暗，低切可改善声音的湿润感和低频的堆积。

Delay

可有可无，因为每首歌速度不同，不可能每首歌都去调速度。

侧链压缩

长混响也可以加个侧链压缩，用人声压混响音量。

乐器注意事项

如果是乐器可以把混响中的 modulation 去掉。

🎹 伴奏轨

EQ / 频率闪避

可以用 EQ 或频率闪避插件适当减少与人声冲突的频段。如 FabFilter Q3、Soundevice.Digital.VoxDucker、Trackspacer。

变调

变调插件不建议使用。一般用酷狗的伴奏功能，或者用全民 K 歌自带变调会方便。

侧链压缩闪避

也可以用侧链压缩做闪避效果，但要注意此时不可用延迟大的变调或其他插件。

📊 总线轨

压缩器 + 限制器

常用插件有 Solid.State.Logic.Native.Bus.Compressor、PSPaudioware.PSP.Xenon、L2、软件自带的 limiter。

响度目标

配合前边人声动态的调整，副歌目标响度建议定在 -12 lufs 附近，峰值建议 -1dB。

DAW 基本操作教学

🎛️ 需要教会客户的操作

  • 人声音量大小调节
  • 伴奏音量大小调节（也可以在音乐软件中调节）。在伴奏软件中音量大小还可以设置快捷键，以达到快速调节目的。
  • 混响大小调节
  • 各个轨道的静音和录制开关
收尾：DAW 工程文件快捷方式放在桌面，方便客户快速打开。

帮客户总结开播前检查事项

①检查声卡指示灯

电脑开机后查看声卡指示灯是否亮了。

请检查声卡 USB 线与电脑是否插好了。

②打开机架和伴奏软件，检查声音

打开机架，然后打开伴奏软件，检查是否能听到自己声音和伴奏。

  • 如果自己声音听不到，请检查机架中麦克风轨道监听是否打开。
  • 如伴奏听不到，请检查机架中伴奏轨道监听是否打开。
③检查伴奏轨道电平

检查机架中伴奏轨道电平是否跳动。

请检查伴奏软件中通道是否正确。

④检查直播伴侣麦克风

打开直播伴侣，检查麦克风图标是否有跳动。

请调整直播伴侣中的麦克风通道。
